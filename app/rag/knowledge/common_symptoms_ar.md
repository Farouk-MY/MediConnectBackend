# الأعراض الشائعة — دليل الاستجواب الطبي بالعربية والتونسية

## صداع / وجع الراس

### أسئلة مهمة للصداع
- الموقع: أمام الرأس، الجوانب، خلف الرأس، كل الرأس، جانب واحد؟
- الطابع: نبضي، ضاغط، حاد، خفيف؟
- المدة: قداش يدوم كل مرة؟
- المحفزات: التوتر، الضوء القوي، أكلة معينة، قلة النوم؟
- أعراض مصاحبة: غثيان، تقيء، تغير في النظر، تصلب الرقبة، سخانة؟
- الشدة: من 1 لـ 10، يأثر على حياتك اليومية؟

### بالتونسي — كيفاش يحكي المريض التونسي
- "راسي يدور" = دوخة مع صداع
- "راسي يوجعني برشة" = صداع شديد
- "راسي ثقيل" = صداع مع إرهاق
- "عيني تزغرد" = اضطراب بصري مع صداع
- "راسي يحرق" = صداع حاد مع حرارة
- "نحس بضغط في راسي" = صداع ضاغط
- "ما نجمش نفتح عيني من الوجع" = صداع شديد مع حساسية للضوء

## وجع الصدر / دولور في الصدر

### أسئلة مهمة
- الموقع: وسط الصدر، الجانب الأيسر، الأيمن، يمتد للذراع/الفك/الظهر؟
- الطابع: ضاغط، حارق، طاعن، مثل الضغط؟
- المحفزات: مجهود بدني، أكل، تنفس، توتر؟
- أعراض مصاحبة: ضيق تنفس، عرق، غثيان، خفقان؟

### بالتونسي
- "صدري يعصرني" = ألم ضاغط في الصدر
- "قلبي يخبط" = خفقان القلب
- "ما نجمش نتنفس" = ضيق في التنفس
- "نحس بحرقة في صدري" = حرقة في الصدر
- "صدري يوجعني كي نتنفس" = ألم مع التنفس

## وجع الكرش / البطن

### أسئلة مهمة
- الموقع: فوق، تحت، يمين، يسار، وسط؟
- الطابع: تشنجات، حادة، حرقة، نفخة؟
- العلاقة بالأكل: قبل، وقت، بعد الأكل؟
- أعراض مصاحبة: غثيان، تقيء، إسهال، إمساك، دم في البراز؟

### بالتونسي
- "كرشي توجعني" = ألم في البطن
- "كرشي منفوخة" = انتفاخ
- "عندي حرقان في المعدة" = حموضة المعدة
- "عندي إسهال" / "كرشي ماشية" = إسهال
- "معدتي تحرق" = حرقة المعدة
- "كرشي تقرقر" = أصوات في البطن
- "ما نجمش ناكل" = فقدان الشهية
- "نتقيأ" / "نرد" = تقيء

## وجع الظهر

### بالتونسي
- "ظهري يكسرني" = ألم شديد في الظهر
- "ما نجمش نوقف" = ألم يمنع الوقوف
- "رجلي تخدر" = تنميل في الساقين
- "عندي ديسك" = انزلاق غضروفي (يفهمها المريض هكا)
- "ظهري مبلوكي" = تصلب في الظهر

## السخانة / الحمى

### بالتونسي
- "عندي السخانة" = حمى
- "جسمي يحرق" = ارتفاع حرارة الجسم
- "عندي البرد" / "نرعش" = قشعريرة
- "حرارتي طالعة" = ارتفاع درجة الحرارة
- "جسمي يوجعني الكل" = آلام عضلية مع حمى

## الكحة / السعال

### بالتونسي
- "عندي كحبة" = سعال
- "نكح بالدم" = سعال مع دم
- "عندي بلغم" = بلغم/مخاط
- "حلقي يحرقني" = احتقان الحلق
- "ما نجمش نبلع" = صعوبة في البلع
- "صوتي راح" = بحة في الصوت

## التعب / الإرهاق

### بالتونسي
- "نحس روحي تعبان/ة" = أشعر بالتعب
- "ما عندي قوة حتى نوقف" = إرهاق شديد
- "ديما نعسان/ة" = نعاس مستمر
- "ما فيا حيل" = لا طاقة
- "جسمي كامل تعبان" = إرهاق عام
- "حاسس/ة روحي ضعيف/ة" = ضعف عام

## الدوخة

### بالتونسي
- "راسي يدور" = دوخة
- "الدنيا تدور بيا" = دوار (فرتيقو)
- "نحس باش نطيح" = إحساس بالإغماء
- "كي نوقف نحس بدوخة" = دوخة وضعية

## مشاكل البول

### بالتونسي
- "نمشي برشة للحمام" = كثرة التبول
- "يحرقني كي نبول" = حرقة بولية
- "لون البول غامق" = تغير لون البول
- "عندي التهاب في البول" = التهاب بولي (المريض يستعمل هالمصطلح)

## وجع المفاصل

### بالتونسي
- "ركبتي توجعني" = ألم الركبة
- "يدي متبلكين" = تصلب اليدين
- "كتفي يوجعني" = ألم الكتف
- "ما نجمش نمشي من الوجع" = ألم يمنع المشي
- "مقاصلي يوجعوني" = ألم المفاصل

## مشاكل الجلد

### بالتونسي
- "جسمي يحكني" = حكة
- "طلعتلي حبوب" = طفح جلدي
- "جلدي أحمر" = احمرار
- "عندي حساسية" = حساسية جلدية
- "طلعتلي حاجة غريبة في جلدي" = آفة جلدية

## مشاكل العين

### بالتونسي
- "عيني حمرا" = احمرار العين
- "عيني تدمع" = دموع
- "ما نشوفش مليح" = ضعف النظر
- "عيني تحكني" = حكة في العين
- "عيني منفوخة" = تورم حول العين

## مشاكل الأذن

### بالتونسي
- "وذني توجعني" = ألم الأذن
- "ما نسمعش مليح" = ضعف السمع
- "عندي طنين في وذني" = طنين الأذن
- "وذني مسدودة" = انسداد الأذن

## القلق والتوتر

### بالتونسي
- "عندي الستريس" = توتر (يستعملوا الكلمة الفرنسية)
- "ما نرقدش بالليل" = أرق
- "عندي لا ديبريسيون" = اكتئاب (الكلمة الفرنسية)
- "نحس روحي مش نورمال" = أشعر بخطب ما
- "قلبي يخبط من الخوف" = قلق مع خفقان
